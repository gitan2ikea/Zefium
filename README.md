# 🔥 ZEFIUM HUB - Roblox Scripts Website

**Zefium Hub** est un site web moderne et premium dédié aux scripts Roblox. Avec un design époustouflant en orange et blanc, des animations 3D fluides et un système de clés unique, ce site offre une expérience utilisateur exceptionnelle.

## 🎨 Caractéristiques

### Design et Animations
- 🌟 **Design Moderne**: Gradient orange/blanc avec animations 3D fluides
- ✨ **Animations Fluides**: Scrolls, parallax et transitions 3D comme orizo.ai
- 🎯 **Responsive**: Compatible avec tous les appareils (desktop, tablet, mobile)
- 🔮 **Effets 3D**: Boîtes 3D qui tournent, étoiles scintillantes et bien plus

### Pages

#### 1. **HOME** (`index.html`)
- Landing page attractive
- Hero section avec animations 3D
- Section des features principales
- Scroll indicators avec animations

#### 2. **SCRIPTS** (`scripts.html`)
- Catalogue complet des scripts Roblox
- 8+ scripts populaires disponibles
- Cartes interactives avec descriptions
- Boutons "OBTENIR LA CLÉ" intégrés

#### 3. **KEYS** (`keys.html`)
- Système de déverrouillage de clés unique
- 12 jeux Roblox populaires
- Intégration Linkvertise (redirection)
- Génération de clés dynamiques et uniques
- Modal pour afficher et copier la clé

## 🚀 Système de Clés

### Fonctionnement
1. L'utilisateur clique sur "DÉVERROUILLER" pour un jeu
2. Redirection vers Linkvertise (simulée)
3. Une fois Linkvertise terminé, une clé unique est générée
4. La clé s'affiche dans un modal
5. L'utilisateur peut copier la clé

### Format des Clés
- Format: `XXXX-XXXX-XXXX-XXXX`
- Caractères: A-Z et 0-9
- Unique pour chaque utilisation
- Stockage local avec localStorage
